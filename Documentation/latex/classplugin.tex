Returns the default excution points. By default it doesn't have any. The following execution points are available inside the rolisz framework:
\begin{DoxyItemize}
\item beforeMatch fires in the router module before trying to match the current URL.
\item afterMatch fires in the router module after it has succesfully matched a URL. If no URL has been matched, it won't fire.
\item hydrateTable fires in the table class if an object has been hydrated from the database.
\item saveTable fires in the table class if an object has been successfully saved to the database.
\item deleteTable fires in the table class if an object has been successfully deleted from the database.
\item compileTemplate fires in the template engine after compiling a template.
