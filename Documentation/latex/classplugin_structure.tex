Registers a plugin. First parameter can be either the name of a plugin, a plugin file or a plugin class itself. Second parameter is optional and speciefies when to execute the plugin if it has to be executed sometime during the execution of rolisz classes. For a list of events, see plugin class. If the plugin constructor requires parameters, you have to instantiate an object and pass that as a parameter to this function.
